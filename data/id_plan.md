# Investigación y Desarrollo (I+D) - Plan 2025-3 / 2026

## Descripción del Área
I+D es el núcleo tecnológico de Unicode. Se encarga de desarrollar software, investigar nuevas tecnologías y mantener la infraestructura digital de la organización.

## Proyectos Principales

### 1. Unicode AI (MVP)
* **Descripción:** Asistente virtual inteligente integrado para facilitar el acceso a la información de la organización.
* **Tecnologías:** Python, FastAPI, LangChain, Google Gemini / OpenAI, Supabase (Vector Store).
* **Estado:** En desarrollo (Fase Backend e Integración RAG).

### 2. Trikaweb
* **Descripción:** Plataforma web principal de Unicode.
* **Objetivo:** Centralizar recursos, eventos y gestión de miembros.

### 3. UniCore
* **Descripción:** Sistema central de gestión (Core) para la integración de servicios internos.

## Metodología de Trabajo
* Uso de metodologías ágiles (Scrum/Kanban) para el seguimiento de proyectos.
* Repositorios en GitHub para control de versiones.
* Capacitaciones internas (Peer Learning) en tecnologías nuevas.
